# AI-Driven Customer Analytics and Personalization Platform

## Problem Statement

Businesses struggle to understand customer behavior and predict future actions from large datasets. Traditional analytics often provide static reports without intelligent insights. This project uses Machine Learning techniques to segment customers, predict purchases, identify churn risk, and generate product recommendations.

---

## Features

- Customer Segmentation using K-Means Clustering
- Purchase Prediction using Random Forest Regression
- Customer Churn Prediction using Random Forest Classification
- Product Recommendation System
- Interactive Flask Dashboard
- Data Visualization and Reporting
